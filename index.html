boh

The smart, simple build tool.

What is boh?

boh is a simple, smart command line build tool with plugins and colors. It's designed for fast, painless prototyping and built with simplicity in mind.

Installation

Install boh via .

npm install -g boh

Usage

Using boh is a simple as running boh in the directory of your project. boh will find the rules needed to build your project by itself.

Build Rules

One of boh's nicest features is the ability to define build rules directly in your files. boh rules are simply YAML at the top of a file in a comment (denoted with !boh). With a set of core rules: build, link and ignore, boh is a powerful system that allows you to interface with the command line tools you know. Every rule thereafter is a plugin that boh will automatically require after an npm install, globally and locally.

•••
style.scss
/*!boh
 * build: sass $this:../build/output.css
 * ignore: ../build/output.css
 * link: ./**/*.scss
 */
import 'login.scss';
import 'header.scss';
import 'footer.scss';

body {
    background: #f04625;
    font-family: 'Helvetica Neue', sans-serif;
    font-size: 16px;

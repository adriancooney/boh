What is boh

boh is a teensy command line tool. When run, it searches through your files looking for build rules and runs them.

Installation

boh is installed via npm.


npm install -g boh

Usage

Using boh is a simple as running boh in the directory of your project. boh will find the rules needed to build your project by itself.
